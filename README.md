# OG Predict Backend (Docker Space)

This Space runs the backend API in a Docker container.

Required environment variables:
- `DATABASE_URL`
- `STORAGE_PRIVATE_KEY`
- `OG_KV_STREAM_ID`
- `OG_KV_NODE_RPC`
- `COMPUTE_PRIVATE_KEY`
- `OG_COMPUTE_API_KEY`
- `ORACLE_PRIVATE_KEY`
- `DEMO_USDC_ADDRESS`
- `TRADING_HUB_ADDRESS`
